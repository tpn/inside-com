ActiveX Template Library for Windows NT 3.51
--------------------------------------------

1. Use PKUNZIP to extract these files:
 	pkunzip -d atl.zip
	pkunzip -d samp.zip
	pkunzip -d docs.zip
2. Run	regsvr32 bin\register.dll
  (this is only necessary for atl.zip).
3. Run	regsvr32 template\multcast.dll
  (this is only necessary for atl.zip).
