Hlink control.

    Properties
  • Hlink - Target URL.
  • Location - Location within the target URL specified.
  • TargetFrame - Target Frame for navigation.
  • Flags - Navigation flags
    • 1 - Navigation required is an internal jump. (HLNF_INTERNALJUMP)
    • 2 - Navigation required in a new browser window. (HLNF_OPENINNEWWINDOW)
  • Mode
    • 0 - HlinkSimpleNavigateToString
    • 1 - HlinkSimpleNavigateToString with NULL pUnk
    • 2 - HlinkNavigateString
    • 3 - HlinkNavigateString with NULL pUnk
    • 4 - HlinkSimpleNavigateToMoniker
    • 5 - HlinkSimpleNavigateToMoniker with NULL pUnk
    • 6 - HlinkNavigateMoniker
    • 7 - HlinkNavigateMoniker with NULL pUnk

    Methods
  • Go - Navigate to the specified URL.
  • Back - Navigate to the previous page if available.
  • Forward - Navigate to the next page if available.
NOTE: HLSIMPLE.DOC available in the specs directory under INETSDK installation explains these functions in detail.
