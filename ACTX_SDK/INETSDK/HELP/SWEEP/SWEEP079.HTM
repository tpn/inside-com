Microsoft Win32 Internet Functions


The Microsoft® Win32® Internet (WinInet) functions provide stand-alone applications with easy access to standard Internet protocols such as Gopher, FTP, and HTTP, abstracting the protocols into a high-level interface that is familiar to Win32 developers.
