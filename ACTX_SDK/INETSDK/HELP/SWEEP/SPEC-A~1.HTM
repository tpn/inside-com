Active Documents

  • OLE Document Objects (DocObjs) This specification describes the interfaces for creating or hosting OLE document objects that can be viewed in arbitrary frames (for example: Web browsers).
  • Simple Hyperlink Navigation This specification is a describes a subset of the ActiveX Hyperlinking specification. It describes hyperlink navigation support for new or existing documents. Application and control developers are recommended to use this subset of the full ActiveX Hyperlinking architecture.
  • ActiveX Hyperlinking This specification describes the complete ActiveX Hyperlinking architecture which provides hyperlinking support for ActiveX-enabled applications or ActiveX controls.
